SigmaWeb 2021-TD Java
------------------------------------------------------------------------------

1/ Thème
-------------------
Projet simple Hibernate/Maven/Mysql

2/ Todo
-------------
- insérer un employé (id, firstName, lastName)
- lire un employé (d'abord inexistant puis existant)
- détruire un employé
    
3/ Résultats attendus
-------------
- l'insertion est accompagnée de trace, on visualise l'insertion dans l'utilitaire 
- de même pour la lecture on constate que l'id lu n'existe pas

    
4/ Notice d'install
-------------
 - créer la DB Mysql
 - créer le fichier POM
 - paramétrer hibernate.cfg (cf tuto)
 
 
5/ Autres solutions, cas dégradé
-------------
 
6/ Bilan
-------------
Avec Hibernate, pas besoin d'écrire directement le code SQL (insert, select...)
